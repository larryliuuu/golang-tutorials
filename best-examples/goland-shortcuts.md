# GoLand IDE Shortcuts

## GO
#### Go fmt
```Shift ⌘ L```

## CODING
#### Show package structure
```Shift ⌘ P```
#### Go to implementation
```Shift ⌘ I```
#### Show Usages
```Shift ⌘ U```
#### Fold / Unfold selection
```⌘ .```
#### Show Context Actions (Fill all fields, ...)
```Alt Enter```
#### Generate
```⌘ N```
#### Smart Code Completion
```Shift ⌘ Space```

## EDITING
#### Refactor
```⌃ t```
#### Comment
```⌃ /```
#### Multi-cursor select / deselect
```⌥``` > Optionally ```double-click``` words
#### Replace all
```⌘ F``` > ```⌃ ⇧ G``` > Deselect anything via ```⌥```

## SEARCHING
#### Search everywhere
```Shift Shift```

&nbsp;&nbsp;&nbsp; Search Types ```⌘ O```

&nbsp;&nbsp;&nbsp; Search Files ```Shift ⌘ O```

&nbsp;&nbsp;&nbsp; Search Symbols ```Shift ⌘ S```

&nbsp;&nbsp;&nbsp; Search Actions ```Shift ⌘ A```

## NAVIGATING
#### Next File / Previous File
```⌃ Tab / ⌃ Shift Tab```
#### Go to line
```⌘ L```

## VIEWING / SHOWING
#### Show recent files
```⌃ e```
#### Show opened file
```Shift ⌘ E```
#### Show Project Explorer
```⌘ 1```
#### Show Git log
```⌘ 2```
#### Show Terminal
```⌘ T```
#### Split and move right
```⌃ S```
#### Unplit all
```⌃ U```
#### Show Problems
```⌘ 6```

## TESTING
#### Run/Debug
```⌃ R / ⌃ D```
#### Stop
```⌥ ⌘ S```
#### Step Over
```⌃ O```
#### Step Into
```⌃ I```
#### Step Out Of
```Shift ⌃ I```
#### Evaluate Expression
```⌃```

## LEGEND
```⌘``` Command

```⌃``` Ctrl

```⌥``` Option/Alt

```⇧``` Shift

```⇪``` Caps Lock

```fn``` Function

## SETTINGS
#### Keymap
```⌘ ,``` > ```Keymap```
#### Export Keymap Settings
```File``` > ```Manage IDE Settings``` > ```Export Settings``` > ```Keymap```
