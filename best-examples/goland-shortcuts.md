# GoLand IDE Shortcuts


## GO
#### Go fmt
`⇧ ⌘ L`


## CODING
#### Show context actions (fill all fields, ...)
`⌥ Enter`
#### Go to declaration
`⌘ D`
#### Show usages
`⌘ U`
#### Go to implementation
`⌘ I`
#### Show package structure
`⌘ P`
#### Type hierarchy
`⌃ H`
#### Fold / unfold selection
`⌘ .`
#### Generate
`⌘ N`
#### Show parameter info
`⇧ ⌘ P`
#### Code completion: basic
`⌃ Space`
#### Code completion: type matching
`⇧ ⌘ Space`
#### Code completion: complete current statement
`⇧ ⌘ Enter`


## EDITING
#### Refactor
`⌃ T`
#### Comment
`⌃ /`
#### Multi-cursor select / deselect
`⌥` > Optionally `double-click` words
#### Replace all
`⌘ F` > `⌃ ⇧ G` > Deselect anything via `⌥`


## SEARCHING
#### Search everywhere
`⇧ ⇧`
#### Search Types
`⌘ O`
#### Search Files
`⇧ ⌘ O`
#### Search Symbols
`⇧ ⌘ S`
#### Search Actions
`⇧ ⌘ A`


## NAVIGATING
#### Move caret forward to next paragraph
`⌃ N`
#### Move caret backward to previous paragraph
`⌃ B`
#### Move caret to matching brace
`⌃ M`
#### Next file / previous file
`⌃ Tab / ⌃ ⇧ Tab`
#### Cursor Back / forward
`⌘ [` / `⌘ ]`
#### Navigation bar
`⌘ ↑`
#### Exit into Editor window
`Esc`
#### Go to line
`⌘ L`


## SHOWING
#### Show recent files
`⌘ E`
#### Show opened file
`⇧ ⌘ E`
#### Show Structure
`⌘ T`
#### Show Project Explorer
`⌘ 1`
#### Show Git Log
`⌘ 2`
#### Show Debug
`⌘ 2`
#### Show Run
`⌘ 4`
#### Show Problems
`⌘ 5`
#### Show Terminal
`⌘ 6`
#### Show TODO
`⌘ 7`
#### Show Find
`⌘ 8`
#### Show Services
`⌘ 9`
#### Split and move right
`⌃ S`
#### Un-split all
`⌃ U`
#### Show Problems
`⌃ P`


## TESTING
#### Run/Debug
`⌃ R / ⌃ D`
#### Stop
`⌥ ⌘ S`
#### Step over
`⌃ O`
#### Step into
`⌃ I`
#### Step out of
`⇧ ⌃ I`
#### Evaluate expression
`⌃ E`

## Version Control
#### VCS pop-up
`⌃ V`
#### Commit
`⌘ K`
#### Commit and push
`⌥ ⌘ K`


## LEGEND
`⌘` Command `⌃` Ctrl `⇧` Shift `⌥` Option/Alt
